Please rename this R sharing template folder, as well as the .rproj here and the script in the "R" folder.

Then when you open the project, make sure to open your renamed script from your new R folder. 
You can navigate to this with the pane on the bottom right

Use the folders in the following way:
Data - Store raw datasets in here
Documentation - Store methods documents in here, descriptions of the analysis, example analyses that you have followed etc.
Outputs - Store edited/generated datasets here, as well as plots and tables
R - Store R scripts in here.
